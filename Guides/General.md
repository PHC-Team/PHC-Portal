# Revix-Portal

<div align="center">

*A modern Steam depot & manifest management application.*

</div>

## ✨ Features

### 📦 Manifest Library
- Browse and download **100,000+** Steam manifests
- Search by Game ID or name
- Automatic installation
- Download history and live progress

### 🎮 Game Preparation
- Built-in **Goldberg Steam Emulator** integration
- SteamStub unpacking
- DRM detection (including Denuvo)
- Automatic emulator configuration
- ColdClient loader support

### 📚 Library Management
- Manage Steam games and Lua scripts
- List & Grid views
- Image caching
- Search and sorting
- Batch updates

### ☁️ Cloud Saves
Powered by **CloudRedirect**.

- Dropbox
- OneDrive
- Google Drive
- Cross-device game pinning

### 🛠 Integrated Tools
Includes:
- DepotDownloader
- DepotDumper
- Config.vdf Extractor
- Goldberg Token Generator

### 🎨 Customization
- Multiple built-in themes
- Custom theme editor
- UI scaling
- Gradient editor
- Background images

### ⚡ Quality of Life
- Automatic updates
- Windows notifications
- System tray support
- Settings backup & restore

---

## 💻 Requirements

- Windows 10 or Windows 11
- Self-contained executable (.NET not required)
- Internet connection

---

## 🚀 Quick Start

1. Download the latest release.
2. Run **Revix-Portal.exe**.
3. Start downloading and managing your Steam content.
